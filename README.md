# Проект приложения платформы-торговой сети электроники
## Описание:
**Приложение для работы с базой данных, представляющее собой торговую сеть электроники.**
## Цель проекта:
Реализовать приложение, которое представляет собой иерархическую структуру из трех уровней торговой цепочки
## Установка:
1. Клонируйте репозиторий:
```python
https://github.com/PetrMaev/retail_chain.git
```
```python
git@github.com:PetrMaev/retail_chain.git
```
2. Установите зависимости:
```python
poetry install
```
## Наполнение данными базы данных:
Для наполнения данными базы данных используйте фикстуры и кастомные команды.
1. Кастомная команда для заполнения базы данных тестовыми данными:
```python
python manage.py add_data
```
2. Кастомная команда для создания суперпользователя:
```python
python manage.py csu
```
3. Кастомная команда для заполнения базы данных пользователями для тестирования:
```python
python manage.py add_users
```
## Использование:
Для запуска приложения введите в терминале следующую команду:
```python
python manage.py runserver
```
## В проекте реализованы:
1. Регистрация пользователей.
2. Авторизация.
3. Вывод в админ-панели созданных объектов.
4. Фильтрация участников сети в админ-панели по названию города.
5. Возможность с помощью admin-action очистить задолженность у нескольких участников сети.
6. Запрет на редактирование поля "Задолженность перед поставщиком".
7. Доступ к API открыт только для активных пользователей.
8. Пагинация списка звеньев торговой сети.
9. Фильтрация участников сети по определенной стране и уровню в структуре.
